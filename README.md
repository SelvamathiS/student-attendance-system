# student-attendance-system
Web-based application developed using Java and MySQL to manage student attendance, maintain records, and generate attendance reports efficiently.


## Project Description
Developed a simple web-based Student Attendance Management System with separate login modules for Admin, Teacher, and Student. The Admin can add student details, manage users, and view complete attendance records. Students can send attendance requests to teachers, and teachers can approve or reject the requests. The system automatically calculates present percentage, absent details, and generates monthly attendance reports in Excel format.

## Features
- Admin Login
- Teacher Login
- Student Login
- Add Student Details
- Attendance Request System
- Approve / Reject Attendance
- Monthly Attendance Report
- Excel Sheet Generation
- Present & Absent Percentage Calculation

## Technologies Used
- Java
- JSP
- Servlet
- MySQL
- HTML
- CSS
